What's this?

The WikiTranslator is a tool for the automated translation of sentences, words and strings in articles of the Team Fortress Wiki.

It features multiple translation functions which can be fully customizable applied to any article.

What's it for?

Though this tool can be used on any article and page, it was intentionally designed for the translation of item articles, especially cosmetic ones.
Not only are those pages similar in many aspects, but with each major update come up to one hundred new cosmetics whose translation would be tedious.
